Problem Collection

Eight fundamental problems in combinatorial optimization

Set Covering Problem

SCP — 9 Tests

Given a universe U = {1,...,m} and a collection of n subsets with costs, find a minimum-cost sub-collection covering all elements. The greedy algorithm achieves the best possible ln(m)+1 approximation ratio unless P=NP.

min ∑ c_j x_j
s.t. ∑_{j: i ∈ S_j} x_j ≥ 1   ∀ i ∈ U
     x_j ∈ {0, 1}
AlgorithmTypeComplexityDetails
ILP Solver Exact NP-hard Integer linear programming formulation via SciPy HiGHS
Greedy Set Cover Heuristic O(m*n) Iteratively pick most cost-effective set; ln(m)+1 approx (Chvatal, 1979)

Graph Coloring Problem

GCP — 11 Tests

Assign colors to vertices of an undirected graph such that no two adjacent vertices share the same color, minimizing the number of colors used (chromatic number). NP-hard to determine and inapproximable within n^(1-epsilon).

AlgorithmTypeDetails
Greedy Coloring Heuristic Sequential coloring with degree-based ordering (Brelaz DSatur, 1979)

Quadratic Assignment Problem

QAP — 10 Tests

Given n facilities and n locations with flow matrix F and distance matrix D, find an assignment minimizing ∑ F[i,j] * D[pi(i), pi(j)]. One of the hardest combinatorial optimization problems — instances with n > 30 are very challenging. Generalizes the Linear Assignment Problem.

AlgorithmTypeDetails
Greedy QAP Heuristic Greedy construction based on flow-distance interaction (Koopmans-Beckmann, 1957)
Simulated Annealing Metaheuristic Swap neighborhood with Boltzmann acceptance for QAP

Maximum Independent Set

MIS — 11 Tests

Find the largest subset S of vertices such that no two vertices in S are adjacent. Equivalent to Maximum Clique on the complement graph. NP-hard and inapproximable within n^(1-epsilon) for any epsilon > 0.

AlgorithmTypeDetails
Branch and Bound Exact Exact B&B with pruning for maximum independent set
Greedy MIS Heuristic Greedy minimum-degree vertex selection (Boppana & Halldorsson, 1992)

Minimum Vertex Cover

MVC — 13 Tests

Find the smallest subset S of vertices such that every edge has at least one endpoint in S. NP-hard (Karp, 1972), but admits a simple 2-approximation. Complement of Maximum Independent Set.

AlgorithmTypeDetails
Greedy Vertex Cover Heuristic 2-approximation via edge matching (Bar-Yehuda & Even, 1981)

Maximum Clique

MC — 11 Tests

Find the largest complete subgraph (clique) in an undirected graph — a subset where every pair of vertices is connected. NP-hard with no polynomial-time constant-factor approximation unless P=NP.

AlgorithmTypeDetails
Bron-Kerbosch Exact Enumerate all maximal cliques with pivoting; O(3^(n/3)) worst-case (Bron & Kerbosch, 1973)

Maximum Weight Set Packing

SPP — 13 Tests

Given a universe of elements and a collection of weighted subsets, find a maximum-weight subcollection of pairwise disjoint sets. NP-hard; greedy achieves 1/k-approximation where k is the maximum set size.

AlgorithmTypeDetails
Greedy Set Packing Heuristic Weight-density greedy with conflict checking (Hurkens & Schrijver, 1989)

Job Sequencing with Deadlines

1 | d_j | Σw_j U_j — 11 Tests

Select and sequence a subset of jobs on a single machine to maximize total profit, where each selected job must complete before its deadline. The unit-weight case is solvable in O(n log n); the weighted case is NP-hard.

AlgorithmTypeDetails
Greedy Job Sequencing Heuristic Sort by profit, assign to latest available slot (Moore, 1968; Lawler & Moore, 1969)
