Linear Programming

LP with Sensitivity Analysis — 11 Tests

Optimize a linear objective function subject to linear equality and inequality constraints. The workhorse of operations research, solvable in polynomial time. Implementation includes full sensitivity analysis: shadow prices, reduced costs, and allowable ranges for objective coefficients and constraint RHS values.

min c^T x
s.t. A_ub x ≤ b_ub
     A_eq x = b_eq
     l ≤ x ≤ u
AlgorithmTypeComplexityDetails
SciPy HiGHS Solver Exact Polynomial Interior-point and simplex via HiGHS; dual values, sensitivity ranges
Shadow Prices
Dual values showing marginal value of each constraint
Reduced Costs
How much objective improves per unit of non-basic variable
Allowable Ranges
Coefficient ranges preserving optimal basis
Infeasibility Detection
Identifies infeasible and unbounded problems

Quadratic Programming

QP via SciPy Minimize — 11 Tests

Optimize a quadratic objective function subject to linear constraints. Extends linear programming with a positive semidefinite quadratic term. Core method for portfolio optimization, support vector machines, and control systems. Solved via Sequential Least Squares Programming (SLSQP).

min (1/2) x^T Q x + c^T x
s.t. A_ub x ≤ b_ub
     A_eq x = b_eq
     l ≤ x ≤ u
where Q is positive semidefinite
AlgorithmTypeComplexityDetails
SLSQP Solver Exact Polynomial Sequential Least Squares Programming via scipy.optimize.minimize
Convex QP
Guaranteed global optimum for convex objectives (PSD Q)
Bound Constraints
Variable bounds and linear equality/inequality constraints
Portfolio Applications
Markowitz mean-variance optimization as canonical use case
KKT Conditions
Optimality verified via Karush-Kuhn-Tucker conditions
