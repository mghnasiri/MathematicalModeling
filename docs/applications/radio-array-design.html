Radio Telescope Array Configuration

Multi-Objective Facility Location · VLA · SKA · ngEHT

Placing antennas across a landscape is a bi-objective facility-location problem. Maximise coverage of spatial frequencies in the Fourier (uv) plane while minimising the total cable and fibre cost connecting every dish to the correlator. This page lets you explore, compare, and optimise array configurations interactively.

Science Context

How interferometers image the sky

Radio Interferometry

A radio interferometer combines signals from many antennas to synthesise a virtual telescope whose angular resolution equals that of a single dish as wide as the maximum antenna separation. Each pair of antennas (a baseline) measures one spatial-frequency sample in the Fourier domain, called the uv-plane.

As the Earth rotates, each baseline traces an ellipse in the uv-plane over the course of an observation. The set of all ellipses determines which spatial frequencies are sampled. Complete, uniform uv-coverage produces clean images with low sidelobe levels; gaps produce artefacts.

The array configuration therefore directly determines imaging quality. Designers must balance coverage uniformity against practical costs: longer baselines require more cable, more trenching, and higher signal-transport budgets. This trade-off is naturally modelled as a bi-objective optimisation problem.

The Problem

Bi-objective antenna placement

ElementDescription
Decision variables(x₁, y₁), ..., (xₙ, yₙ) — antenna positions on a 2-D site
Objective 1 (max)uv-plane coverage uniformity — fraction of Fourier cells filled
Objective 2 (min)Total cable / fibre length — minimum spanning tree cost
ConflictBetter coverage requires more spread-out antennas, which increases cable cost
CombinatoricsFor ngEHT site selection: C(35, 5) = 749,398 candidate combinations
Mathematical Formulation Maximise f₁ = UV_coverage(x₁,y₁, ..., xₙ,yₙ) Minimise f₂ = MST_cost(x₁,y₁, ..., xₙ,yₙ) subject to xᵢ ∈ [Xmin, Xmax], yᵢ ∈ [Ymin, Ymax] -- site boundaries dᵢⱼ ≥ dmin ∀i≠j -- minimum separation

Why It Is Hard

  • Continuous, non-convex search space in 2N dimensions (antenna positions)
  • uv-coverage depends on Earth rotation synthesis — each baseline generates an elliptical track over time
  • The two objectives are Pareto-conflicting: no single layout is optimal for both
  • Minimum-separation and site-boundary constraints create a non-trivial feasible region
  • Evaluating coverage requires computing N(N-1)/2 baselines at multiple hour angles
