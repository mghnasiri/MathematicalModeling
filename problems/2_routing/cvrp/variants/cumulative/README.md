# Cumulative VRP (CumVRP)

## Problem Definition

Minimize total arrival time at customers (sum of arrival times across all routes) instead of total travel distance. Also known as Minimum Latency VRP or Multi-Vehicle Traveling Repairman.

## Applications

Disaster relief, humanitarian logistics, service routing where customer waiting time matters.
