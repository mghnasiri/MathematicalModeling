# Capacitated p-Median Problem (CPMP)

## Problem Definition

Extends p-Median with facility capacities. Open exactly p facilities and assign customers to minimize total weighted distance, subject to per-facility capacity constraints.

## Complexity

NP-hard (generalizes p-Median).

## Algorithms

| Algorithm | Type | Reference |
|-----------|------|-----------|
| Greedy add | Heuristic | Mulvey & Beck (1984) |
| Teitz-Bart interchange | Heuristic | Mulvey & Beck (1984) |
| Simulated Annealing | Metaheuristic | Lorena & Senne (2004) |
