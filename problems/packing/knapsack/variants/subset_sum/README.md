# Subset Sum Problem (SSP)

## Problem Definition

Given a set of n positive integers and a target T, find a subset summing to exactly T (decision) or as close to T as possible without exceeding it (optimization).

## Complexity

NP-complete (decision). Weakly NP-hard with O(nT) pseudo-polynomial DP.
