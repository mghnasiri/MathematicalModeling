# Multi-Depot Vehicle Routing Problem (MDVRP)

## Problem Definition

Multiple depots with vehicle fleets. Assign customers to depots and build routes minimizing total distance.

## Complexity

NP-hard (generalizes CVRP).

## Algorithms

| Algorithm | Type | Reference |
|-----------|------|-----------|
| Nearest-Depot NN | Heuristic | Cordeau et al. (1997) |
| Simulated Annealing | Metaheuristic | — |
