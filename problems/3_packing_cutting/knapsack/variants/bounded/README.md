# Bounded Knapsack Problem (BKP)

## Problem Definition

Given n item types with values, weights, and upper bounds on copies, maximize total value within capacity.

## Complexity

NP-hard (weakly); O(n*W) pseudo-polynomial DP.

## Algorithms

| Algorithm | Type | Reference |
|-----------|------|-----------|
| Greedy Density | Heuristic | Kellerer et al. (2004) |
| Dynamic Programming | Exact | Kellerer et al. (2004) |
| Simulated Annealing | Metaheuristic | — |
